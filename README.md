## Grupo: Carla Lima (1139025) e Rafaela Davesac (1138820)

### Descrição:
Criamos uma nova coluna informando a frequência relativa de cada tipo de golpe de estelionato. Para isso, aplicamos uma função que converteu o número de ocorrências de cada categoria na pesquisa em sua respectiva proporção dentro do total de respostas.

### Justificativa:
Calcular a frequência relativa dos golpes permite identificar com maior precisão quais tipos de ataques são mais recorrentes. Essa métrica facilita a visualização de padrões e auxilia na interpretação dos dados, destacando quais golpes são mais frequentemente relatados pelos participantes.

### Análises que essa conversão possibilita:
- Identificar quais golpes são mais comuns
- Comparar as proporções entre diferentes tipos de golpes
- Gerar visualizações através de gráficos

### Banco de dados utilizado: Estelionato
### Coluna utilizada: 'Qual foi o último tipo de golpe que você presenciou ou foi vítima?'
